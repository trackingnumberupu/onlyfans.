    Firestore Setup Required

    Cloud Firestore API is not enabled. Follow the steps below to enable it and get full cloud sync across devices.

    Step 1: Enable Cloud Firestore API in Google Cloud

    1. Go to Google Cloud Console → Firestore API
    2. Select project: onlyfans-clone-6ca98
    3. Click Enable

    Step 2: Create Firestore Database

    1. Go to Firebase Console → Firestore Database
    2. Click Create database
    3. Choose Start in test mode (for development)
    4. Select a region close to your users (e.g., nam5/us-central)
    5. Click Enable

    Step 3: Set Firestore Security Rules

    rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId} { allow read, write: if request.auth != null && request.auth.uid == userId; } match /users/{userId}/data/{doc} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }

    Step 4: Verify It Works

    1. After enabling, refresh this page
    2. Sign up or log in again
    3. Your profile data will now sync across all devices!
